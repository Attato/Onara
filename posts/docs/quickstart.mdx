# Quickstart

This section details how to get started in onara.

## Installation

Clone Onara app from our official repository <a href="https://github.com/Attato/Onara" target="_blank">here.</a>

Or use the command:

<CodeBlock language="bash" code={`git clone https://github.com/Attato/Onara.git`} showLineNumbers={false}/>

Then type this code into the console to install all Onara dependencies:

<CodeBlock language="shell" code={`npm install`} showLineNumbers={false}/>


You can learn more about project dependencies in the Contributing section.

## Launching

<CodeBlock language="shell" code={`npm run dev`} showLineNumbers={false}/>

Type this code in the console to launch the Onara application locally.
