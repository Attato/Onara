# Code Block

The CodeBlock component is a reusable React component that displays syntax-highlighted code and provides a copy-to-clipboard functionality. It utilizes the Prism component from the react-syntax-highlighter library to render the code with a specified language and style.

## Usage

Import the CodeBlock component and use it in your React application as follows:

{' '}

<CodeBlock language="tsx" code={`import CodeBlock from '@/components/CodeBlock';

const MyComponent = () => {
	return(
		<div>
			<CodeBlock language="javascript" code={'console.log("Hello world!")'} />
		</div>
	); 
};

export default MyComponent;`} />

What you see above is the CodeBlock component. Here is the result you will get if you follow the example:

<CodeBlock language="javascript" code={'console.log("Hello world!")'} />

## Props
The CodeBlock component accepts the following props:

<CodeBlock language="md" code={
`| Prop     	| Type   	| Default Value                   	| Description                    	|
|----------	|--------	|---------------------------------	|--------------------------------	|
| language 	| string 	| md                              	| The language of the codeblock. 	|
| code     	| string 	| The majestic empty block of ... 	| The content of the codeblock.  	|`} />
     
If you do not pass any values to the component and use it, as in this example:

<CodeBlock language="tsx" code={`<CodeBlock />`} />

You will get this result:

<CodeBlock />
