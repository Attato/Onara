# Icon Wrapper

The Icon Wrapper component is a wrapper for SVG icons. It allows you to customize the width, height, stroke width, and fill color of the icon.

## Usage

Import the Icon Wrapper and use it in your React application as follows:

<CodeBlock language="tsx" code={`import IconWrapper from '@/components/IconWrapper';

const MyComponent = () => {
	return(
		<div>
         <IconWrapper>
            <path
                strokeLinecap="round"
                strokeLinejoin="round"
                d="M15.59 14.37a6 6 0 01-5.84 7.38v-4.8m5.84-2.58a14.98 14.98 0 006.16-12.12A14.98 14.98 0 009.631 8.41m5.96 5.96a14.926 14.926 0 01-5.841 2.58m-.119-8.54a6 6 0 00-7.381 5.84h4.8m2.581-5.84a14.927 14.927 0 00-2.58 5.84m2.699 2.7c-.103.021-.207.041-.311.06a15.09 15.09 0 01-2.448-2.448 14.9 14.9 0 01.06-.312m-2.24 2.39a4.493 4.493 0 00-1.757 4.306 4.493 4.493 0 004.306-1.758M16.5 9a1.5 1.5 0 11-3 0 1.5 1.5 0 013 0z"
            />

};

## Props
The Icon Wrapper component accepts the following props:


<CodeBlock language="md" code={`| Prop        	| Type      	| Default Value 	| Description                           	|
|-------------	|-----------	|---------------	|---------------------------------------	|
| width       	| number    	| 16            	| The width of the icon wrapper.        	|
| height      	| number    	| 16            	| The height of the icon wrapper.       	|
| strokeWidth 	| number    	| 2             	| The stroke width of the icon wrapper. 	|
| fill        	| string    	| 'none'        	| The fill color of the icon wrapper    	|
| children    	| ReactNode 	| children      	| The content of the svg tag            	|`} />
