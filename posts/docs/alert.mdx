# Alert

The Alert component is a React functional component that displays an alert message with an optional icon. It supports different types of alerts such as warning, attention, and success. The component uses CSS modules for styling.

## Usage

Import the Alert component and use it in your React application as follows:

<CodeBlock language="tsx" code={
`import Alert from '@/components/Alert';

const MyComponent = () => {
	return(
		<div>
			<Alert type="warning" text="This is a warning alert" />
			<Alert type="attention" text="This is an attention alert" />
			<Alert type="success" text="This is a success alert" />
			<Alert text="This is a default alert" />
		</div>
	); 
};

export default MyComponent;`
} />

## Props
The Alert component accepts the following props:


<CodeBlock language="tsx" code={
`export type AlertType = 'warning' | 'attention' | 'success' | null;

type AlertProps = {
	type?: AlertType;
	text?: string;
};`
} />

<CodeBlock language="md" code={
`| Prop 	| Type           	| Default Value      	| Description               	|
|------	|----------------	|--------------------	|---------------------------	|
| type 	| string \| null 	| attention          	| The type of the alert.    	|
| text 	| string         	| Alert without text 	| The content of the alert. 	|`
} />
