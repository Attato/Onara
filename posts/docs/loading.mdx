# Loading
The Loading component is a customizable spinner that can be used to indicate a loading state in a React application.

## Usage
To use the Loading component, follow these steps:

Import the component:

<CodeBlock language="tsx" code={`import Loading from '@/components/Loading';`} />

Place the component in your JSX code:

<CodeBlock language="tsx" code={`<Loading />`} />
<br />
<Loading />

## Props

The Loading component accepts the following props:
<CodeBlock language="md" code={`| Prop   	| Type   	| Default Value 	| Description                                                  	|
|--------	|--------	|---------------	|--------------------------------------------------------------	|
| width  	| number 	| 24            	| The width of the spinner in pixels.                          	|
| height 	| number 	| 24            	| The height of the spinner in pixels.                         	|
| color  	| string 	| '#0082f5'     	| The color of the spinner. Accepts any valid CSS color value. 	|`} />

## Example

Here's an example of using the Loading component with custom props:

<CodeBlock language="tsx" code={`<Loading width={32} height={32} color="cornflowerblue" />`} />
<br/>
<Loading width={32} height={32} color="cornflowerblue" />

You can use component loading to display the loading process

<button disabled style={{display: 'flex', alignItems: 'center', gap: '8px', height: "fit-content", padding: '12px 16px', backgroundColor: '#0f0f0f', border: '1px solid #232323'}}>Loading <Loading width={16} height={16} /></button>

With the Loading component, you can easily display a loading spinner with customizable width, height, and color, providing a visual indication of an ongoing process or waiting state in your application.
