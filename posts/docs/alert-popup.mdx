# Alert Popup

The AlertPopup template is a React functional component that displays an alert inside a popup. It is composed of the Alert and Popup components.

## Usage

To use the AlertPopup template, follow these steps:

Import the necessary dependencies:

<CodeBlock language="tsx" code={`import AlertPopup from '@/components/_Templates/AlertPopup';`} />

Place the component in your JSX code:

<CodeBlock language="tsx" code={`<AlertPopup />`} />

Result:

<AlertPopup />

## Props:

<CodeBlock language="md" code={
`| Prop          	| Type      	| Default Value      	| Description                           	|
|---------------	|-----------	|--------------------	|---------------------------------------	|
| buttonContent 	| ReactNode 	| Blank popup        	| The button content of the alert popup 	|
| title         	| string    	| Untitled           	| The title of the alert popup.         	|
| alertType     	| string    	| attention          	| The type of the alert popup.          	|
| alertText     	| string    	| Alert without text 	| The content of the alert popup        	|`} />

## Example
Here's an example usage of the AlertPopup component:

<CodeBlock language="tsx" code={
`<AlertPopup
    title="My Alert"
    alertType="success"
    alertText="This is a success message."
    buttonContent={
        <button style={{
            backgroundColor: "#0f0f0f",
            border: "1px solid #232323",
            borderRadius: "6px",
           padding: "12px 16px",
           display: "flex",
           alignItems: "center",
           gap: "4px",
           color: "#fefefe"
        }}>
            Example
            <GiftIcon width={16} height={16}/>
        </button>
    }
/>`} />

Result:

<AlertPopup
    buttonContent={
        <button style={{
            backgroundColor: "#0f0f0f",
            border: "1px solid #232323",
            borderRadius: "6px",
            padding: "12px 16px",
            display: "flex",
            alignItems: "center",
            gap: "4px",
            color: "#fefefe"
        }}>
            Example
            <GiftIcon width={16} height={16}/>
        </button>
    }
    title="My Alert"
    alertType="success"
    alertText="This is a success message."
/>

In this example, a popup with the title "My Alert" is displayed, containing a success alert with the message "This is a success message."
