# Popup

The Popup component is a reusable React component that provides a popup overlay with a title, content, and close button. It uses the framer-motion library for animations.

## Usage
To use the Popup component, follow these steps:

Import the necessary dependencies:

<CodeBlock language="tsx" code={`import Popup from '@/components/Popup';`} />

Place the component in your JSX code:

<CodeBlock language="tsx" code={`<Popup />`} />
<br />

<Popup />

## Props

The Loading component accepts the following props:
<CodeBlock language="md" code={`| Prop          	| Type      	| Default Value                     	| Description              	|
|---------------	|-----------	|-----------------------------------	|--------------------------	|
| buttonContent 	| ReactNode 	| <button>Blank popup</button>      	| The button of the popup. 	|
| title         	| string    	| Untitled                          	| The title of the popup.  	|
| children      	| ReactNode 	| <p>The legend states that ...</p> 	| The content of the popup 	|`} />

## Conclusion
The Popup component provides a flexible and reusable way to create popup windows in React applications. It allows you to easily control the popup's visibility and customize its content and appearance.
