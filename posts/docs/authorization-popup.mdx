# Authorization Popup

The Authorization Popup template is a reusable component used for displaying a popup window with options for user authorization using different services. It is implemented using Next.js and NextAuth.js.

<CodeBlock language="tsx" code={`<AuthorizationPopup />`}/>

<AuthorizationPopup />


## Props 

<CodeBlock language="md" code={`| Prop          	| Type      	| Default Value                	| Description                                   	|
|---------------	|-----------	|------------------------------	|-----------------------------------------------	|
| buttonContent 	| ReactNode 	| <button>Blank popup</button> 	| The button content of the authorization popup 	|
| title         	| string    	| Untitled                     	| The content of the authorization popup        	|`} />
