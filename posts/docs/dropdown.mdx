# Dropdown

The Dropdown component is a reusable React component that provides a dropdown functionality with a toggle button. It uses the motion and AnimatePresence components from the Framer Motion library for smooth animations.

## Usage

Import the Dropdown component and use it in your React application as follows:

<CodeBlock language="tsx" code={`import Dropdown from '@/components/Dropdown';

const MyComponent = () => {
	return(
		<div>
			<Dropdown />
		</div>
	); 
};

export default MyComponent;`} />

Result: 
<Dropdown />

## Props
The Dropdown component accepts the following props:

<CodeBlock language="md" code={
`| Prop          | Type      | Default Value                    | Description                      |
|---------------|-----------|----------------------------------|----------------------------------|
| buttonContent | ReactNode | 'Dropdown'                       | The label (tsx) of the dropdown. |
| children      | ReactNode | <button>One, Two, Three</button> | The content of the dropdown.     |`} />
